Book Shelf

Online books

This website can be used as an online Bookshelf where the users can add, delete and edit information regarding books. The webpage is developed using html5, css, angularjs and bootstrap. Created by Ravali Bolem. I recently completed my master's from University of Missouri - Kansas City in the field of Computer Science with a GPA of 3.8

Basic operations of this website are:


Books

View the list of books

Add

Add a book to the list



Remove

Remove a book from the book shelf

Update

Change the details of a book
